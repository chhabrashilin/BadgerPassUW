Scripts
=======

Planned utility scripts (create after frontend/program are generated):
- `generate-qr.ts`: Generate a PNG QR that deep-links to `/checkin/[eventPubkey]`.
- `seed-demo.ts` (optional): Create a couple of sample events for demo.

Suggested run commands:
- `npm run tool:qr -- --event <EVENT_PUBKEY> --out ./qrs/event.png`
